MCEngineArtificialIntelligenceApi Documentation

Constructor

MCEngineArtificialIntelligenceApi(Plugin plugin)

Initializes the AI API for your plugin, loads AddOns, DLCs, and registers default models from config.yml

// Example usage
MCEngineArtificialIntelligenceApi api = new MCEngineArtificialIntelligenceApi(this);

checkUpdate()

checkUpdate(String gitPlatform, String org, String repository, String token)

Checks for updates from GitHub, GitLab, etc.

api.checkUpdate("github", "MCEngine", "artificialintelligence", null);

getPlugin()

Returns the Bukkit Plugin instance.

Plugin myPlugin = api.getPlugin();

registerModel()

registerModel(String platform, String model)

Registers an AI model for a specific platform (runs once per unique model).

api.registerModel("openai", "gpt-4");

getResponse()

getResponse(String platform, String model, String message)

Gets a response from the AI using the specified platform and model.

String reply = api.getResponse("openai", "gpt-4", "Tell me a joke!");
